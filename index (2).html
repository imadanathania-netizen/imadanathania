Mengenal Jenis Celana

Pelajari karakteristik dan perbedaan antara Celana Kain dan Celana Jeans sebelum menggunakan fitur deteksi AI.

👔

Celana Kain

Formal
  • Umumnya digunakan untuk acara formal, perkantoran, dan kegiatan resmi
  • Material lebih ringan, tipis, dan memberikan tampilan yang rapi dan profesional
  • Permukaan kain halus dengan jahitan yang presisi dan terstruktur
  • Tersedia dalam berbagai warna netral: hitam, abu, navy, cokelat
  • Cocok dipadukan dengan kemeja, jas, atau blazer untuk penampilan elegan
PolyesterWoolViscoseLinenCotton blend
👖

Celana Jeans

casual
  • Terbuat dari bahan denim yang kuat dan tahan lama untuk pemakaian jangka panjang
  • Lebih casual dan cocok untuk aktivitas sehari-hari, santai, dan rekreasi
  • Tekstur kain lebih tebal dengan pola tenunan diagonal yang khas
  • Identik dengan warna biru denim, namun tersedia juga hitam, putih, dan abu
  • Mudah dipadukan dengan kaos, kemeja flannel, atau jaket untuk gaya casual
DenimCottonStretch denimRaw denim

Teknologi yang Digunakan

SmartPants AI dibangun menggunakan teknologi machine learning terkini melalui Teachable Machine dari Google, memungkinkan klasifikasi gambar yang akurat langsung di browser.

Teachable Machine TensorFlow.js HTML5 Canvas MediaDevices API JavaScript ES6+ CSS3 Animations

SmartPants AI adalah aplikasi web berbasis kecerdasan buatan yang dikembangkan sebagai tugas mata kuliah Artificial Intelligence. Website ini mengimplementasikan machine learning dalam klasifikasi jenis pakaian.

Model AI dilatih menggunakan Google Teachable Machine dengan dua kelas: Celana Kain dan Celana Jeans. Proses inferensi berjalan sepenuhnya di browser tanpa mengirim data ke server eksternal.
